# Vision

Eye of Horus aims to become a research-grade geometry-native programming language and toolchain.

The long-term vision includes:

- a precise language specification;
- a Rust reference implementation;
- visual debugging for spatial programs;
- educational examples for alternative computation models;
- an RFC process for community language design;
- editor and language-server support;
- a book and website for learners and contributors.

The near-term goal is narrower: define the smallest executable language core that can be parsed, simulated, tested, and explained without exaggeration.
