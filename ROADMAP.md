# Roadmap

This roadmap separates implemented work from planned work. Dates are intentionally omitted until maintainers can commit to schedules responsibly.

## Phase 0: Public Research Repository

Status: in progress.

- repository structure;
- Apache-2.0 license;
- contribution and governance documents;
- white paper v0.1 draft;
- formal specification skeleton;
- minimal Rust workspace.

## Phase 1: Language Core

Status: planned.

- lexer and parser;
- AST and diagnostics;
- spatial matrix representation;
- deterministic pulse simulation;
- clear value encoding rules;
- first executable `.eoh` examples;
- test suite for parser and core semantics.

## Phase 2: Tooling Alpha

Status: planned.

- CLI interpreter prototype;
- visualizer prototype;
- VS Code syntax highlighting;
- initial LSP diagnostics;
- RFC-backed syntax stabilization.

## Phase 3: Research and Community

Status: planned.

- formal semantics review;
- computational power analysis;
- educational materials;
- standard library experiments;
- community working groups.
