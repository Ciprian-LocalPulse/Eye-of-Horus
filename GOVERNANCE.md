# Governance

Eye of Horus begins as a founder-led research project by Ciprian Stefan Plesca. The governance goal is to move toward transparent community participation as the project matures.

## Maintainer Responsibilities

- protect project honesty and research integrity;
- review contributions;
- enforce the code of conduct;
- keep implemented, planned, and speculative work clearly labeled;
- steward the RFC process.

## Decision Process

Minor documentation and tooling changes may be accepted through pull requests. Language semantics, syntax, memory model, runtime behavior, and standard-library design require RFC discussion before implementation.

## Future Structure

Future working groups may cover compiler, specification, tooling, documentation, security, and education.
