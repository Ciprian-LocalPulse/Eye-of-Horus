# FAQ

## Is Eye of Horus production ready?

No. It is research-stage software.

## What exists today?

A professional repository, draft specification documents, RFC process, examples, and a minimal Rust scaffold.

## What does not exist yet?

A complete parser, compiler, VM, standard library, package manager, LSP, VS Code extension, and stable syntax.

## Is the phi-pi memory model secure?

No. The formula is deterministic and public. It may be studied as a non-linear layout idea, not used as a security boundary.

## Is Eye of Horus Turing-complete?

Not proven. This is an explicit research question.

## Why Rust?

Rust offers memory safety, strong tooling, and performance characteristics suitable for a future interpreter and geometric runtime.
